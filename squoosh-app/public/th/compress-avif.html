บีบอัด AVIF ออนไลน์ — โคเดกรุ่นถัดไป

เข้ารหัสใหม่หรือบีบอัดไฟล์ AVIF ซ้ำด้วยตัวห่อ libavif รอบโคเดก AV1 ที่คุณภาพการรับรู้ที่ตรงกัน AVIF มักจะเล็กกว่า WebP 20–30% และเล็กกว่า JPEG 50%+

AVIF คืออะไรและทำไมฉันควรสนใจ?

AVIF (AV1 Image File Format) เป็นคอนเทนเนอร์ภาพนิ่งรอบโคเดกวิดีโอ AV1

AVIF เล็กกว่าจริงเท่าไหร่ในทางปฏิบัติ?

ในชุดภาพถ่ายผลิตภัณฑ์ทั่วไป การเปลี่ยนจาก JPEG คุณภาพ 80 เป็น AVIF คุณภาพ 50 จะลดขนาดไฟล์ลงประมาณ 50%

การสนับสนุนเบราว์เซอร์และเครื่องมือ

AVIF รองรับใน Chrome 85+, Firefox 93+, Edge 121+ และ Safari 16+

รูปแบบอื่นที่คุณอาจต้องการ

สำหรับความโปร่งใสเต็มที่โดยไม่มีการเลื่อนสี PNG แบบไม่สูญเสียยังคงเป็นตัวเลือกที่ปลอดภัย
