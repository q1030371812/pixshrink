Mengapa GIF begitu besar dan bagaimana kita memperkecilnya?

GIF menggunakan kompresi LZW pada palet terindeks 256 warna. Ini adalah format tahun 1989 dan tidak dirancang untuk jenis rekaman layar warna penuh yang dilemparkan orang ke dalamnya saat ini.

Berapa lebih kecil yang harus saya harapkan?

Pada GIF rekaman layar 480×270, 8 detik pada 15 fps, gifsicle biasanya mengurangi file 30–60% tanpa perubahan visual.

Bisakah saya menghilangkan frame atau mengubah loop?

Ya. Panel opsi memungkinkan Anda mengatur frame rate target.

Kapan beralih format

GIF adalah format animasi paling kompatibel yang ada. Untuk web dan sebagian besar aplikasi, WebP animasi mengalahkan GIF dalam ukuran dan biaya baterai.

Pertanyaan yang sering diajukan

Apakah kompresi GIF kehilangan frame?

Mode default adalah tanpa loss — setiap frame tetap ada.

Mengapa GIF saya masih 3 MB?

GIF dengan gerakan penuh yang panjang secara fundamental besar. Mengonversi ke WebP animasi biasanya memberikan file 4–8× lebih kecil.

Bisakah mengubah GIF ke format video?

Ya — halaman yang sama dapat mengenkode ulang input GIF sebagai WebP animasi.

Apakah animasi masih diputar di setiap browser?

Ya. GIF adalah format animasi yang paling didukung secara luas.
